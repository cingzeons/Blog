1. 珠峰博客接口文档 #

  • 登录
    • post /api/users/singup 注册
    • post /api/users/signin 登录
    • get /api/users/signout 登出
  • 分类列表
    • get /api/categories 列表
    • post /api/categories 增加
    • put /api/categories/:id 编辑
    • delete /api/categories/:id 删除
  • 文章列表
    • get /api/articles 列表
    • post /api/articles 增加
    • put /api/articles/:id 编辑
    • delete /api/articles/:id 删除
    • get /api/articles/pv/:id 增加pv
    • post /api/articles/:id 增加评论

4. 文章 #

4.1 文章列表 #

4.1.1 请求名称 #

/api/articles

4.1.2 请求方法 #

GET

4.1.3 请求参数 #

放在URL地址中的查询字符串里

  • "pageNum":1, //显示当前显示的页
  • "pageSize":5, //显示每页条数是5
  • "keyword":"a" //关键字
pageNum=1&pageSize=5&keyword=a

    "data": {

                        "_id": "5ae7217f176d5a17507d6ae9",
                        "content": "评论内容1",
                        "user": "5ae70c24a8003f0cdcfc4ebc"

                    {
                        "createAt": "2018-04-30T14:06:09.560Z",
                        "_id": "5ae722d18463811c606f4643",
